BETA VERSION

RPG Clicker – V0.00.01

RPG Clicker is a browser-based incremental game focused on progression, upgrade mechanics, and relaxing gameplay. Built with HTML, CSS, JavaScript and Firebase integration.
Core Features
Account System
Secure login & logout using Firebase authentication. User session stored locally for smooth experience.
Upgrade Mechanism
Progressive click scaling system including additive (+) and multiplier (x) upgrades.
Shop System
Purchase power boosts and unlock new potential.
Achievement System
Track milestones and reward long-term players.
User Status Panel
Display real-time statistics and player progression.
Background Customization Future Update
Personalize the click area with selectable themes.
